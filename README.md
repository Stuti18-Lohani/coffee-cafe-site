 ## **Coffee Cafe Site** - Premium UI Experience
  **Live Demo Here:-**(https://stuti18-lohani.github.io/coffee-cafe-site/)

Welcome to the **Coffee Cafe Site**, a modern, responsive, and visually appealing landing page designed for a premium coffee shop experience. This project was built to showcase clean UI/UX principles and responsive web design.

 **Project Overview**
The goal of this project was to create a seamless user interface that captures the essence of a cozy cafe. It features a professional layout, high-quality imagery integration, and interactive elements.

 **Tech Stack**
- **HTML5**: Semantic structure for better SEO and accessibility.
- **CSS3**: Custom styling with Flexbox/Grid for a modern layout.
- **Google Fonts**: Integrated for premium typography.
- **Font Awesome**: For crisp and scalable icons.

 **Key Features**
- **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop screens.
- **Hero Section**: An engaging landing area with a clear Call-to-Action (CTA).
- **Menu Gallery**: A beautifully structured display of coffee varieties.

- **Smooth Navigation**: Easy-to-use navbar for better user flow.
- **Contact Integration**: A dedicated section for cafe location and inquiries.
